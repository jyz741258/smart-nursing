### ✅ 3. 音效管理系统 (`utils/audioManager.ts`)

**程序化生成音效（无需音频文件）**：
- ✅ **Success** - 清脆上行音（523Hz → 1046Hz）
- ✅ **Error** - 低沉下行音（200Hz → 100Hz + 低通）
- ✅ **Warning** - 颤音提示（440Hz 颤音）
- ✅ **Info** - 短促高频（800Hz）
- ✅ **Click** - 轻微反馈（600Hz）
- ✅ **Notification** - 大三和弦（C-E-G）
- ✅ **Complete** - 上行琶音（C大调）

**特点**：
- 使用 Web Audio API 实时合成
- 音量控制 + 全局开关
- 状态持久化（localStorage）

### ✅ 10. 全局页面转场 (`App.vue` 升级)

**新特性**：
- 🎬 **6种转场动画**：
  - `fade-transform` - 淡入 + 缩放位移
  - `slide-up-fade` - 向上滑动（进入深层级）
  - `slide-down-fade` - 向下滑动（返回上级）
  - `slide-left` - 横向滑动
  - `scale-bounce` - 弹性缩放（模态页）
  - `blur-fade` - 模糊聚焦（iOS 风格）

- 🧠 **智能路由感知**：
  ```typescript
  // 根据路由深度自动选择转场
  if (toDepth > fromDepth) → slide-up-fade
  if (toDepth < fromDepth) → slide-down-fade
  else → fade-transform
  ```

- 🚀 **GPU 加速**：`will-change`, `backface-visibility`, `perspective`

## 🎛️ 音效控制 API

### 全局控制
```typescript
import { audioManager } from '@/utils/audioManager'

// 播放音效
audioManager.play('success')
audioManager.play('error')
audioManager.play('warning')
audioManager.play('info')
audioManager.play('click')
audioManager.play('notification')
audioManager.play('complete')

// 设置音量（0-1）
audioManager.setVolume(0.5)

audioManager.setEnabled(false)

// 获取状态
const enabled = audioManager.isEnabled()
const volume = audioManager.getVolume()
```

```vue

import { inject } from 'vue'

const audioManager = inject('audioManager')

const handleSuccess = () => {
  audioManager.play('success')
```

## 🎯 性能优化建议

### 1. 按需加载动画
```typescript
// 对于不常用的动效，使用动态导入
const { CounterAnimation } = await import('@/components/CounterAnimation.vue')
```

### 2. 减少重绘
- 所有动画使用 `transform` + `opacity`（触发 GPU）
- 避免 `width/height` 动画
- 使用 `will-change` 提示浏览器

### 3. 降低移动端功耗
```scss
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

### 4. 音效预加载
- Web Audio API 需要用户交互后激活
- 建议在登录页添加"点击开启音效"引导

## 💡 设计理念

### 1. **深色优先**
- 默认深色模式（更现代、更护眼）
- 浅色模式作为可选

### 2. **动效克制**
- 每个动画都有明确目的
- 避免过度动画导致疲劳
- 支持减弱动态效果（无障碍）

### 3. **性能第一**
- 优先使用 transform/opacity
- GPU 加速
- 合理使用 will-change

### 4. **一致性**
- 全局统一的缓动曲线
- 统一的颜色变量
- 统一的动画时长
