### 阶段 5：AI 聊天室动效（10分钟）

#### 修改 `src/views/AIChat.vue`

```vue
<template>
  <div class="chat-container">
    <!-- 消息列表 -->
    <div ref="messagesContainer" class="messages-wrapper">
      <ChatMessage
        v-for="(msg, index) in messages"
        :key="msg.id"
        :content="msg.content"
        :is-user="msg.role === 'user'"
        :timestamp="msg.timestamp"
        :status="msg.status"
        :quick-replies="msg.quickReplies"
        :delay="index * 100" // 阶梯延迟
        @quick-reply="handleQuickReply"
      />
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import ChatMessage from '@/components/ChatMessage.vue'
import { audioManager } from '@/utils/audioManager'

const messages = ref([
  {
    id: 1,
    role: 'assistant',
    content: '你好！我是智能护理助手，有什么可以帮您？',
    timestamp: new Date(),
    quickReplies: ['查看今日服务', '预约护理', '健康咨询']
  }
])

const handleQuickReply = (reply: string) => {
  // 播放提示音
  audioManager.play('click')

  // 添加用户消息
  messages.value.push({
    id: Date.now(),
    role: 'user',
    content: reply,
    timestamp: new Date()
  })

  // AI 回复
  setTimeout(() => {
    messages.value.push({
      id: Date.now() + 1,
      role: 'assistant',
      content: `好的，正在处理您的请求："${reply}"`,
      timestamp: new Date()
    })
  }, 500)
}
</script>
```

## 🐛 调试与问题排查

### 问题 1：音效不播放

**原因**：浏览器要求用户交互后才能播放音频

**解决**：
```javascript
// 在用户首次点击时初始化
document.addEventListener('click', async () => {
  await audioManager.init()
}, { once: true })
```

### 问题 2：主题切换不生效

**检查**：
```javascript
console.log(document.documentElement.getAttribute('data-theme'))
// 应该是 "dark" 或 "light"
```

**强制刷新**：
```bash
# 清除 localStorage
localStorage.removeItem('smart-nursing-theme')
```

### 问题 3：3D 卡片效果卡顿

**优化**：
- 减少同时倾斜的卡片数量
- 降低 `maxTilt` 值（默认 15° → 改为 5°）
- 移动端自动禁用

### 问题 4：导入 Excel 失败

**原因**：缺少依赖

**解决**：
```bash
cd frontend
npm install xlsx jspdf jspdf-autotable
```

### 问题 5：数字滚动不显示

**检查**：确保 `value` 是数字类型
```javascript
// ❌ 错误
value: "123"

// ✅ 正确
value: 123
```

## 📞 技术支持

### 常见问题

**Q: 为什么音效不播放？**
A: 浏览器策略要求用户交互后才能播放音频。确保用户在页面上点击过至少一次。

**Q: 如何关闭所有动画？**
A: 在 `theme.scss` 中设置 `--duration-fast: 0s` 或在浏览器 DevTools 中开启 "Disable animations"

**Q: 移动端效果不佳？**
A: 部分高级效果（3D 倾斜）在移动端已自动禁用，可以通过媒体查询进一步优化。

**Q: PDF 导出中文乱码？**
A: 需要配置中文字体。建议使用 `jspdf` 的字体插件或转换为图片。
